# NHANES Analysis: Obesity and Hypertension

## Overview

This project examines the association between obesity and hypertension using NHANES data.

## Methods

* Logistic regression (R and Stata)
* Adjusted for age and sex
* Complete-case analysis

## Key Findings

* Obesity associated with higher odds of hypertension (OR ≈ 1.78)
* Age positively associated with hypertension
* Results consistent across R and Stata

## Files

* `scripts/analysis_r.R` → R analysis
* `scripts/analysis_stata.do` → Stata analysis
* `output/regression_results.csv` → Model results

## Tools Used

* R (glm, haven, broom)
* Stata
* NHANES dataset
